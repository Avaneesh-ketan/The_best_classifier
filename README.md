#the_best_classifier
This code is the demonstration of the Classifier for a loan repayment status
We consider the following attributes 
1) Loan status
2) Principal amount 
3) Terms
4) Effective date
5) Due date
6) Age 
7) Education 
8) Gender
Usage of Algorithms : 
1) KNN
2) Decision Trees
3) SVM's
4) Logistic Regression
